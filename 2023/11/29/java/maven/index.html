Tomcat插件

使用Tomcat插件发布部署并执行war工程的时候，需要使用启动命令，启动命令为: tomcat7:run.命令中的tomcat7是插件命名，由插件提供商决定。run为插件中的具体功能。

Tomcat是可运行插件，必须要通过命令来运行控制

常见命令

mvn install

本地安装，包含编译、打包、安装到本地仓库

mvn clean

清除已编译信息，删除工程中的target目录

常用的命令在IDEA都有可视化界面可以操作
