SpringBoot基础

框架介绍

SpringBoot主要解决的是在微服务架构下简化配置（有快速配置）、前后端分离、快速开发

优点：

  • 提供了快速启动入门
  • 开箱即用、提供默认配置
  • 内嵌容器化web项目
  • 没有冗余代码生成和XML配置要求

模板引擎

模板引擎： 如 Thymeleaf, FreeMarker

有嵌套和解析的过程，先加载静态页面，在静态页面上添加一些标记，模板引擎的内核会根据添加的标记动态渲染数据
