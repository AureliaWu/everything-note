Git命令

常用命令

git clone xxx(远程仓库地址) 克隆远程仓库至本地

git branch 查看本地分支

git branch -r 查看远程分支

git branch <分支名> 创建本地分支

git status 查看当前状态

git add . 将所有文件添加到暂存区

git commit am "代码提交备注" 提交代码并添加注释

git push 将分支推送至远程

git pull 拉取远程分支

一套代码关联多个远程仓库

git remote -v
